# ZAP

## Introduction

ZAP, or Zip Archive Packaging, is really two things.

It is the file format in which Tribblix packages are distributed, and the
name of the cli tool to manage packaging and other administrative tasks on
a Tribblix system.

The documentation here describes how it works.

* [Packages](Packages.md)
* [Repositories](Repositories.md)
* [Catalogs](Catalog.md)
